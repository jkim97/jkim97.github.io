Ruby Hashes vs. JavaScript Objects

Both Ruby Hashes and JavaScript Objects have key/value pair. In order to access value of the hash by hashname[:key]. On the other hand, you could access object in the JS by putting var_name.key.

Ruby classes vs. JavaScript constructor functions

Both constructor and classes create an object. Both needs initialize; difference is class needs def to initialize the instance variable.

Looping in Ruby vs. Looping in JavaScript

Both have looping method. JavaScript uses 'for' for looping, when Ruby uses 'while' or 'until' to loop the code.

True and False

Ruby uses '==' while JS uses '==='.
