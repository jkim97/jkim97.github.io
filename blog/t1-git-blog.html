Version control keep track of history of changes made to file. By doing so, if file was lost or damaged, we trace it back by version control. Also, it helps multiple people to work on same project as a group. Git is a type of version control.

Git traces all the changes made to file, until git commit is commanded. By typing command 'git status' you can see what kind of change was made. Program would suggest if you would like to keep change or return file to previous state. git reset is a way to delete all the changes made to the file.

One of the benefit of storing code to GItHub is group project. GitHub provide Hub to share a code, which could be worked by different people at different time without interfering other's work.
